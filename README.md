#  Todo Docker App
A simple todo application built with Python and docker.

 ## Files 
 todo.py - python todo application 
 dockerfile - Docker configuration to run the app 
 Docker - commands.md - Docker command practiced
 
  ## How to Run
  Build the image : docker build -t todo-app .
  Run the container:  docker run -it todo-app

  ## Features
  Add todos
  View todos
  Remove todos

  ## Skills Practiced 
  python programming 
  Docker build and run
  docker logs, exec, stop, start 
  Git and Github
